This manuscript now uses one direct proof spine, not a recursive architecture.

\subsection*{Proof spine}
\begin{enumerate}
  \item \textbf{Define the canonical object:}
  \[
  \Phi(q):=\Gamma(q)(1-q)\zeta(q).
  \]

  \item \textbf{Define balance:}
  $q$ is balanced exactly when $\Phi(q)=0$.

  \item \textbf{State two theorem targets:}
  Target~A is strip zero--balance correspondence (\cref{thm:target-correspondence});
  Target~B is strip critical-line exclusion (\cref{thm:target-uniqueness}).

  \item \textbf{Deduce RH conditionally:}
  if Targets~A and~B both hold, RH follows by \cref{prop:rh-reduction-main-spine}.
\end{enumerate}

\subsection*{Status}
Steps 1--2 are definitions. Steps 3--4 are open theorem work.
The argument attempts to establish RH through this route, but RH is not claimed as proved.
