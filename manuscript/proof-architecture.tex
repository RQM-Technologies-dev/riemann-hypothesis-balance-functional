This section records the proof spine used in the manuscript under review.

\begin{enumerate}
  \item Define one object:
  \[
  \Phi(s):=\Gamma(s)(1-s)\zeta(s).
  \]

  \item Use the strip correspondence:
  for $0<\RePart(s)<1$,
  \[
  \Phi(s)=0\iff\zeta(s)=0.
  \]

  \item Isolate one core theorem target:
  \[
  \Phi(s)=0\Longrightarrow\RePart(s)=\tfrac12
  \quad (0<\RePart(s)<1).
  \]

  \item Conclude RH conditionally:
  if Step 3 is proved, RH follows.
\end{enumerate}

\noindent
Steps 1--2 are established setup.
Step 3 is the unresolved burden.
