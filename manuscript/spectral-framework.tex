This section records the simplified framework used in the main manuscript.

\subsection*{Canonical object}
The main text is centered on one function:
\[
\Phi(q):=\Gamma(q)(1-q)\zeta(q).
\]
Balance is defined only by $\Phi(q)=0$.

\subsection*{Reason for this scope}
The manuscript under review is prepared for mathematical audit.
Using one explicit object keeps the proof obligations short and visible:
\begin{enumerate}
  \item identify strip zeros via $\Phi(q)=0$;
  \item exclude off-line strip points via $\Phi(q)=0\Rightarrow \RePart(q)=\tfrac12$.
\end{enumerate}

\subsection*{Secondary role of operator language}
Operator or resolvent interpretations are optional heuristics in this revision.
They are not part of the main theorem chain and are not used to define balance.

\begin{remark}[Status boundary]
This simplification is organizational, not a completed proof step.
The two theorem targets above remain open.
\end{remark}
